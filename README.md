# Rize-Clone

Master project repository for the Rize-clone. This repo aggregates the sub-projects as git submodules:

- `rize-desktop` — desktop client
- `rize-mobile` — mobile client
- `rize-backend` — backend services

## Getting started

Clone with submodules:

```
git clone --recurse-submodules <repo-url>
```

Or initialize submodules after a normal clone:

```
git submodule update --init --recursive
```
